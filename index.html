3. Cargar boundary comunal

Sube un GeoJSON Polygon/MultiPolygon, Feature o FeatureCollection. Se usará para validar jurisdicción y centrar el mapa.

4. Cargar Unidades Vecinales / sectores

Sube un GeoJSON FeatureCollection. El sistema buscará propiedades como sector_code, NOMBRE, UV_CARTO, unidad o barrio.

5. Catálogo maestro App Vecino

Define las categorías disponibles para todas las comunas. Luego cada ADMIN municipal decide cuáles muestra, en qué orden y con qué nombre local.
