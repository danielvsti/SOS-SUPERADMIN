SUPER ADMIN

Alta multi-comuna: centros de control, administrador municipal, geocerca y Unidades Vecinales.

Acceso VS&TI

Entrar como SUPER ADMIN

Solo usuarios con rol SUPER_ADMIN pueden crear comunas y primer administrador municipal.

1. Crear / actualizar Centro de Control

Usa formato de código tipo CC-QUILPUE, CC-VINA, CC-CONCON.

2. Crear primer ADMIN municipal

Este usuario queda amarrado al centro seleccionado. Después podrá crear operadores, resolutores y vecinos de su comuna desde SOS-ADMIN.

3. Cargar boundary comunal

Sube un GeoJSON Polygon/MultiPolygon, Feature o FeatureCollection. Se usará para validar jurisdicción y centrar el mapa.

4. Cargar Unidades Vecinales / sectores

Sube un GeoJSON FeatureCollection. El sistema buscará propiedades como sector_code, NOMBRE, UV_CARTO, unidad o barrio.
